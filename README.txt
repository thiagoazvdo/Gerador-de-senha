# Projeto Simples de Gerador de Senha

Bem-vindo(a) esse projetinho simples que tive a ideia de realiza-lo durante uma aula de tópicos
avançados de programação onde pretendo refatorar com operações lambda e stringBuilder.

## Como executar o programa

Antes de executar a aplicação, certifique-se de ter instalado em sua máquina o Java
(Versão 17 utilizada no desenvolvimento)

1. Faça o clone do repositório para a sua máquina local:

2. Acesse o diretório do projeto

3. Execute a classe GeradorDeSenha e insira os seguintes valores desejados no console: 
	3.1 ~ Quantidade de números desejados na senha que será gerada. 
	
	3.2 ~ Quantidade de letras desejadas na senha. 
	
	3.3 ~ Quantidade de caracteres especiais desejados para a senha.
